# LifeOS v13 — Today Screen

Стабильная версия на базе v12 Stability Center. Главный экран теперь является центром дня: кольца, следующее действие, живой активный интервал, быстрые отметки и compact iPhone-first сценарий.

## Что добавлено в v13

- Новый блок **Today Screen** на главном экране.
- Живой таймер интервала: работа / спорт / саморазвитие.
- Завершённый таймер автоматически сохраняется в журнал.
- Быстрые действия: закрыть план, тренировка, книга, курс, свой вариант.
- Блок «Следующее действие» с локальной логикой антихаоса.
- Более удобная мобильная сетка под iPhone PWA.
- Сохранён весь v12 Stability Center: диагностика, кэш, бэкапы, восстановление, миграции.

## Публикация

Загрузи содержимое архива в корень GitHub Pages репозитория. Если на iPhone стояла старая PWA — удали иконку, открой сайт в Safari, обнови 2–3 раза и добавь заново.
