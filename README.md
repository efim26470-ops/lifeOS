# LifeOS v11 — v8 Stable Base + Complete Rings

Эта сборка сделана от последней стабильной ветки **v8 Rings Edition**, а не от сломанных v9/v10. Новые функции добавлены поверх рабочей архитектуры без урезания старого функционала.

## Что внутри

- PWA для GitHub Pages без сервера и базы данных.
- Выбор графика при первом запуске и в настройках.
- Работа, спорт, саморазвитие и свои варианты.
- Кольца дня в стиле Apple Fitness.
- Новые недельные кольца.
- Умный планировщик дня.
- Вечерний отчёт в стиле Apple Journal.
- История дней отдельной вкладкой.
- Конструктор своих категорий: название, иконка, тип, цель в минутах, участие в кольцах.
- Настраиваемые 3 кольца главного экрана.
- Настройка виджетов главного экрана.
- Свои быстрые шаблоны.
- Спорт-дневник, саморазвитие, книги, курсы, проекты, Ableton, языки и свой вариант.
- Финансы, отчёты, CSV/JSON, Pomodoro, проекты GitHub Pages.
- Бесконечные достижения и XP.
- Темы iOS / Apple Fitness / Apple Music / Apple Journal / Liquid Glass.
- Safe-area адаптация под iPhone PWA, челку, Dynamic Island и нижний home indicator.

## Установка на GitHub Pages

1. Распакуй архив.
2. Загрузи **содержимое архива** в корень репозитория.
3. Включи GitHub Pages: `Settings → Pages → Deploy from branch → main / root`.
4. После публикации открой сайт.

## Важно при обновлении старой PWA на iPhone

Если раньше была установлена битая версия:

1. Удали старую иконку LifeOS с экрана Домой.
2. Открой сайт в Safari.
3. Обнови страницу 2–3 раза.
4. В приложении нажми `Настройки → Очистить кэш PWA`.
5. Снова добавь на экран Домой.

## Хранение данных

Данные хранятся локально: `localStorage`, запасной режим через `sessionStorage/cookie/memory`, а также резерв в `IndexedDB`, когда доступен. Для безопасности периодически делай `Экспорт JSON`.
