Настройки и резервная копия

Как опубликовать на GitHub Pages

  1. Загрузи все файлы из архива в корень репозитория.
  2. Открой Settings → Pages.
  3. Выбери Branch: main, Folder: /root.
  4. После публикации открой сайт и добавь LifeOS на главный экран.

Добавить LifeOS на iPhone

Открой сайт в Safari, нажми кнопку «Поделиться», затем выбери «На экран Домой». После этого LifeOS будет запускаться как отдельное приложение.
