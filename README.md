# LifeOS v12 — Stability Center

Стабильная PWA-сборка на базе рабочей v8/v11. Новые большие функции не вырезаны: работа, спорт, саморазвитие, кольца, график, темы, Liquid Glass, журнал, история, отчёты, ачивки и PWA-режим сохранены.

## Что добавлено в v12

- PWA-центр обслуживания в настройках.
- Диагностика `localStorage`, `sessionStorage`, cookies, IndexedDB, Service Worker и Cache Storage.
- Кнопка «Починить PWA без удаления данных».
- Кнопка «Сбросить интерфейс» без удаления записей.
- Скачивание диагностического JSON.
- Автоматический резерв в IndexedDB раз в несколько часов.
- Версионность данных: `appVersion`, `dataVersion`.
- Защитный экран ошибки, чтобы приложение не падало в пустую страницу.
- Новый cache-name service worker, чтобы GitHub Pages не держал старую сломанную сборку.

## Как загрузить на GitHub Pages

1. Распакуй архив.
2. Загрузи содержимое папки в корень репозитория.
3. Не загружай саму папку целиком, только файлы внутри.
4. Открой сайт через GitHub Pages.
5. На iPhone удали старую иконку LifeOS с экрана «Домой», открой сайт в Safari и добавь заново.

## Если снова держится старая версия

Открой LifeOS → Настройки → PWA-центр обслуживания → «Починить PWA без удаления данных». Потом обнови страницу и заново добавь PWA на экран «Домой».
