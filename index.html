Публикация

  1. Загрузи содержимое архива в корень репозитория.
  2. Открой Settings → Pages.
  3. Выбери Branch: main, Folder: /root.
  4. Открой опубликованный сайт в Safari/Chrome и установи как PWA.

Что хранится локально

Записи времени, отметки работы, привычки, цели, заметки, настроение, XP и настройки. Сервер, база данных и внешние API не используются.

Добавить LifeOS на iPhone

Открой сайт в Safari, нажми «Поделиться», затем выбери «На экран Домой». После этого LifeOS будет запускаться как отдельное приложение.
