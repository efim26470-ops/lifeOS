# LifeOS v15 — Modules

Статическая PWA-сборка для GitHub Pages.

Основа: стабильная v12 + Today Screen v13 + Active Timer v14.

## Новое в v15

- отдельный модуль **Книги**;
- отдельный модуль **Курсы**;
- прогресс страниц, уроков, статусов и заметок;
- чтение и занятия автоматически засчитываются в журнал и кольцо саморазвития;
- история чтения и занятий;
- ачивки за страницы, книги, уроки и курсы;
- отчёт LifeOS теперь включает книги и курсы;
- сохранены работа, спорт, саморазвитие, проекты, таймеры, темы, Liquid Glass, PWA-диагностика, бэкапы и настройки графика.

## Публикация

Распакуй архив и загрузи содержимое в корень репозитория GitHub Pages.

После обновления на iPhone лучше удалить старую иконку PWA, открыть сайт в Safari, обновить страницу 2–3 раза и добавить на экран «Домой» заново.
