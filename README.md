# LifeOS v10 Stable

Статическая PWA-версия для GitHub Pages. Все данные хранятся локально. В этой сборке `index.html` содержит критические CSS/JS внутри себя, чтобы страница открывалась даже при проблемах с отдельными файлами или кэшем.

## Установка на GitHub Pages
1. Распакуй архив.
2. Загрузи файлы в корень репозитория.
3. Включи GitHub Pages.
4. На iPhone после обновления старой версии открой: Настройки → Очистить кэш PWA, затем закрой PWA и открой заново.

## Что внутри
- выбор графика на первом запуске и в настройках;
- кольца работы, спорта и саморазвития;
- свои категории и шаблоны;
- спорт, саморазвитие, фокус, Journal-история, аналитика, бесконечные ачивки;
- темы iOS / Apple Fitness / Apple Music / Apple Journal;
- настройка Liquid Glass и анимаций;
- safe-area под iPhone PWA.
