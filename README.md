# LifeOS — финальная PWA-версия со спортом

**LifeOS** — статическое PWA-приложение для GitHub Pages. Работает без сервера, базы данных и внешних API. Все пользовательские данные сохраняются локально в браузере через `localStorage`; для переноса используется экспорт/импорт JSON.

## Рабочий график

Период уже встроен в приложение:

- **3 июля 2026 — 31 августа 2026**;
- будни: **10:00–17:00**;
- суббота: **10:00–14:00**;
- воскресенье: выходной;
- всего: **51 рабочий день / 330 часов**.

## Что добавлено в расширенной версии

### Работа и антихаос

- отдельный режим **Работа**;
- расчёт плана, факта, остатка и отставания;
- прогноз среднего времени на оставшийся рабочий день;
- проблемные дни: пустые и частичные;
- автоматический расчёт потерянного рабочего времени;
- быстрый зачёт полного рабочего дня;
- снятие авто-зачёта;
- копирование записей вчерашнего дня.

### День

- режим **день открыт / день закрыт**;
- настроение дня;
- заметка дня;
- запись любого интервала времени;
- вариант **Другое** со своим названием;
- отдельные свои варианты: документы, встречи, покупки, творчество, музыка, семья, друзья и любые пользовательские категории;
- быстрый ввод своего варианта прямо из формы интервала;
- визуальный таймлайн дня;
- подсчёт времени по категориям.

### Календарь

- календарь продуктивности в стиле GitHub contribution graph;
- статусы дней: полный, частичный, пустой, будущий, выходной;
- фильтры: все / рабочие / пустые / частичные;
- карточка каждого дня с прогрессом.

### Журнал

- все записи времени в одном списке;
- фильтр по дате;
- фильтр по типу;
- поиск по заметке и категории;
- дублирование и удаление записей;
- экспорт CSV.

### Цели, привычки, фокус и спорт

- цели с прогрессом;
- автоматическая цель по рабочим часам;
- автоматическая цель по серии дней;
- привычки и повседневные таски;
- Pomodoro-таймер;
- сохранение фокус-сессий в журнал.

### Спорт

- отдельная вкладка **Спорт**;
- запись тренировок, прогулок, бега, зала, растяжки, футбола, велосипеда, бассейна и домашней тренировки;
- свой вид спорта через **Свой спорт**;
- интенсивность: лёгкая, средняя, высокая, жёсткая;
- необязательный ввод калорий;
- быстрые спортивные шаблоны;
- статистика: всего спорта, сегодня, за 7 дней, калории, любимая активность, количество видов активности;
- спортивные записи попадают в общий журнал, CSV, JSON и HTML-отчёт;
- достижения за спорт и движение.

### Деньги

- ставка за час;
- бонус за полный рабочий день;
- потеря/штраф за пропущенный час;
- денежная цель;
- прогноз заработка к 31 августа.

### Проекты

- отдельная панель GitHub Pages-проектов;
- статус проекта;
- прогресс;
- дедлайн;
- ссылка на репозиторий;
- ссылка на опубликованный сайт;
- заметки по проекту.

### Отчёты и бэкапы

- экспорт JSON;
- импорт JSON;
- экспорт CSV;
- генерация итогового HTML-отчёта;
- скачивание отчёта отдельным файлом;
- отчёт теперь включает спортивные часы, количество активностей и калории;
- пустой шаблон данных.

### Достижения и темы

- XP и уровни;
- достижения за рабочие дни, фокус, бэкапы, заметки и проекты;
- темы: системная, светлая, тёмная, Midnight, Aurora, **Стукач**;
- PWA-иконки для iPhone, Windows и macOS;
- offline-кэш через Service Worker.

## Публикация на GitHub Pages

1. Распакуй архив.
2. Загрузи все файлы в корень репозитория.
3. Открой **Settings → Pages**.
4. Выбери:
   - Source: `Deploy from a branch`;
   - Branch: `main`;
   - Folder: `/root`.
5. Сохрани настройки и открой ссылку GitHub Pages.

## Установка как приложение

### iPhone / iPad

1. Открой сайт в Safari.
2. Нажми **Поделиться**.
3. Выбери **На экран Домой**.
4. Нажми **Добавить**.

### Windows / macOS

1. Открой сайт в Chrome или Edge.
2. Нажми кнопку установки в адресной строке.
3. Подтверди установку.

## Важно

Данные локальные. Если очистить данные сайта, браузер или `localStorage`, записи исчезнут. Перед переносом на другое устройство или перед чисткой кэша используй **Экспорт JSON**.
