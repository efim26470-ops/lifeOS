# LifeOS v9 — Complete Rings Edition

Статическая PWA для GitHub Pages. Работает без сервера и базы данных: данные хранятся локально в браузере.

## Что внутри

- Главный экран «Сегодня» в стиле Apple Fitness / Apple Journal.
- Три настраиваемых кольца: работа, спорт, саморазвитие или свои категории.
- Недельные кольца.
- Умный планировщик дня и кнопка «Принять план».
- Вечерний отчёт с настроением и заметкой.
- Запись работы, спорта, саморазвития и любого своего варианта.
- Конструктор категорий: название, тип, иконка, цвет, дневная цель.
- Спортивный дневник: тренировки, интенсивность, калории.
- Саморазвитие: проекты, книги, курсы, практика, языки, Ableton и свой вариант.
- Фокус-таймер: 25/5, 50/10, 90/20 и свой режим.
- История дней в стиле Apple Journal.
- Аналитика «что съедает время».
- Итоговый HTML-отчёт.
- Быстрые шаблоны и свои шаблоны.
- Бесконечные ачивки и уровни.
- Темы: iOS Light/Dark, Midnight, Aurora, Apple Fitness, Apple Music, Apple Journal, iOS 7, iOS 13, iOS 17 Pro, iOS 18 Liquid, Aqua, Graphite, iPod, Стукач.
- Настройки Liquid Glass: прозрачность, размытие, анимации.
- iPhone PWA safe-area: челка, Dynamic Island, нижний home indicator.

## Публикация на GitHub Pages

1. Распакуй архив.
2. Загрузи файлы в корень репозитория.
3. Включи GitHub Pages: Settings → Pages → Deploy from branch.
4. Открой сайт в Safari/Chrome.
5. Для iPhone: Safari → Поделиться → На экран «Домой».

## Обновление старой версии

После загрузки новой версии на GitHub Pages открой LifeOS → Настройки → Сбросить кэш PWA. Данные не удаляются, сбрасывается только кэш интерфейса.
