# LifeOS PWA v8 — Rings Edition

Статическое PWA-приложение для GitHub Pages. Сервер, база данных и внешние сервисы не нужны.

## Что нового в v8

- Кольца выполнения целей в стиле Apple Fitness:
  - работа;
  - тренировки;
  - саморазвитие.
- Быстрая отметка работы, спорта и саморазвития с главного экрана.
- Отдельный модуль «Саморазвитие»:
  - проекты;
  - книги;
  - курсы;
  - практика;
  - конспекты;
  - языки;
  - кодинг;
  - Ableton;
  - свой вариант.
- Саморазвитие учитывается в журнале, отчётах, CSV/JSON, XP и бесконечных ачивках.
- Отдельная цель саморазвития в день, настраивается в разделе «Настройки → Визуальный стиль».
- Расширенные темы оформления:
  - системная;
  - iOS Light;
  - iOS Dark;
  - Midnight;
  - Aurora;
  - Apple Fitness;
  - Apple Music;
  - Apple Journal;
  - iOS 7 Frost;
  - iOS 13;
  - iOS 17 Pro;
  - iOS 18 Liquid;
  - Classic Aqua;
  - Graphite;
  - iPod;
  - Стукач.
- Настройка Liquid Glass:
  - сила прозрачности;
  - размытие;
  - уровень анимаций.
- Новые PWA-иконки.
- Service Worker обновлён до v8, чтобы GitHub Pages меньше держал старую версию в кэше.

## Установка на GitHub Pages

1. Распакуй архив.
2. Загрузи все файлы в корень репозитория.
3. Включи GitHub Pages: `Settings → Pages → Deploy from branch`.
4. Открой сайт в Safari/Chrome/Edge.
5. На iPhone: Safari → Поделиться → На экран «Домой».

## Важно при обновлении старой версии

После загрузки новой версии открой приложение и нажми:

**Настройки → Очистить кэш PWA**

Потом полностью закрой PWA на iPhone и открой заново.

## Хранение данных

LifeOS хранит данные локально в браузере. Поддерживаются:

- localStorage;
- IndexedDB-резерв;
- sessionStorage/cookie fallback;
- экспорт JSON;
- импорт JSON;
- CSV-экспорт журнала.

Перед заменой версии рекомендуется сделать JSON-бэкап.
