# LifeOS PWA

**LifeOS** — локальная PWA-панель для контроля работы, дня, спорта, денег, привычек, проектов и отчётов. Проект полностью статический и подходит для GitHub Pages.

## Что внутри

- Первый запуск с выбором рабочего периода.
- Настраиваемый график по дням недели.
- Цель по тренировкам в неделю.
- Журнал интервалов с собственными вариантами времяпрепровождения.
- Отдельный модуль спорта.
- День открыт / день закрыт.
- Таймлайн дня.
- Календарь продуктивности.
- Антихаос: подсказки, отставание, прогноз.
- Финансовый модуль.
- Проекты GitHub Pages.
- Экспорт JSON / CSV / HTML-отчёта.
- Бесконечные достижения.
- PWA-иконки для iPhone, Windows, Android, macOS.
- Offline-режим через service worker.

## Публикация на GitHub Pages

1. Распакуй архив.
2. Загрузи файлы в корень репозитория.
3. Включи GitHub Pages: `Settings → Pages → Deploy from branch`.
4. Открой сайт на iPhone через Safari.
5. Нажми `Поделиться → На экран «Домой»`.

## Важно после обновления старой версии

Если на iPhone продолжает открываться старая верстка:

1. Открой LifeOS.
2. Перейди в `Настройки → Очистить кэш PWA`.
3. Закрой PWA из переключателя приложений.
4. Открой заново.

В v7 service worker уже обновлён так, чтобы следующие релизы подтягивались быстрее.

## Хранение данных

Основное хранение: `localStorage`. Если браузер блокирует его, LifeOS использует запасные варианты и дублирует данные в `IndexedDB`. Для переноса между устройствами используй экспорт JSON.
