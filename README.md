# LifeOS v14 — Active Timer

Статическая PWA-версия для GitHub Pages. Основа: стабильная ветка v13/v12/v8-base, без удаления старых функций.

## Что добавлено в v14

- Отдельная вкладка **Таймеры**.
- Живой таймер для:
  - работы;
  - спорта;
  - саморазвития;
  - перерыва;
  - своего варианта.
- Пауза и продолжение активного интервала.
- Быстрые пресеты: работа, спорт, проект, книга, курс, перерыв.
- Заметка к активной сессии.
- Автоматическое сохранение завершённой сессии в журнал.
- История последних активных сессий.
- Today Screen показывает активный таймер и ведёт на вкладку «Таймеры».
- Обновлён service worker: отдельный cache-name v14.

## Публикация на GitHub Pages

1. Распакуй архив.
2. Загрузи содержимое архива в корень репозитория.
3. Включи GitHub Pages из ветки `main` / root.
4. На iPhone удали старую иконку LifeOS, открой сайт в Safari, обнови страницу 2–3 раза и добавь PWA заново.

## Важно

Если на iPhone открывается старая версия, зайди в настройки LifeOS и нажми очистку PWA-кэша. Данные хранятся локально, перед жёстким сбросом лучше скачать JSON-бэкап.
