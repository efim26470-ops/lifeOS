# LifeOS PWA v16.0.1 — Profiles & Reports Hotfix

Статическая PWA для GitHub Pages. Версия v16 построена поверх v15 Modules и сохраняет прежние функции: работа, спорт, саморазвитие, кольца, таймеры, книги, курсы, проекты, темы, Liquid Glass, диагностика PWA и локальные бэкапы.

## Новое в v16

- Система профилей: **Рабочий режим**, **Учёба**, **Лето**, **Интенсив**, **Минимализм**.
- У каждого профиля могут быть свой график, цели, кольца, тема, виджеты и быстрые шаблоны.
- Красивый HTML/PDF-ready отчёт.
- Кнопки отчётов: скачать HTML, CSV, JSON и скопировать текст.
- Улучшенные темы как визуальные системы: iOS 6 Aqua, iOS 7 Frost, iOS 13 Dark, iOS 17 Pro, iOS 18 Liquid, Apple Fitness, Apple Music, Apple Journal, Graphite, Midnight, Aurora.
- Режим **Мини** для iPhone PWA: Работа / Спорт / Саморазвитие / Быстрая запись / Закрыть день.
- Service Worker обновлён на отдельный кэш v16.

## Установка на GitHub Pages

1. Распакуй архив.
2. Загрузи **содержимое архива** в корень репозитория.
3. Включи GitHub Pages для ветки `main`.
4. На iPhone после обновления удали старую иконку PWA, открой сайт в Safari, обнови страницу 2–3 раза и добавь заново на экран «Домой».

Все данные хранятся локально в браузере. Перед крупными обновлениями делай экспорт JSON.


## Hotfix 16.0.1

- Исправлена критическая ошибка `ringLabel is not defined`, из-за которой приложение могло показывать экран восстановления.

- Старые данные мигрируют через legacy-ключи и не удаляются.
