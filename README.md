# LifeOS PWA

**LifeOS** — статическое PWA-приложение для GitHub Pages. Работает без сервера, базы данных и внешних API. Все данные сохраняются локально в браузере через `localStorage`.

## Что внутри

- рабочий период: **3 июля 2026 — 31 августа 2026**;
- график работы:
  - будни — **10:00–17:00**;
  - суббота — **10:00–14:00**;
  - воскресенье — выходной;
- рабочий день как базовый повседневный таск;
- запись любых интервалов времени;
- вариант **«Другое»** со своим названием времяпрепровождения;
- календарь работы с состояниями: выполнено / частично / пусто / выходной;
- журнал времени с фильтрами, поиском, дублированием и удалением записей;
- заметка дня и настроение дня;
- привычки и ежедневный чек-лист;
- цели с прогрессом;
- Pomodoro / фокус-таймер;
- достижения, XP и уровни;
- аналитика по категориям, последним 14 дням, плановым и фактическим часам;
- экспорт JSON и CSV;
- импорт JSON;
- светлая, тёмная, системная, Midnight и Aurora темы;
- PWA-иконки для iOS/Windows/macOS;
- offline-режим через Service Worker.

## Публикация на GitHub Pages

1. Создай репозиторий на GitHub.
2. Загрузи все файлы из архива в корень репозитория.
3. Открой **Settings → Pages**.
4. В разделе **Build and deployment** выбери:
   - Source: `Deploy from a branch`;
   - Branch: `main`;
   - Folder: `/root`.
5. Открой опубликованную ссылку.
6. На iPhone открой сайт через Safari → «Поделиться» → «На экран Домой».

## Важно

Данные не синхронизируются между устройствами автоматически. Для переноса используй **Экспорт JSON** на старом устройстве и **Импорт JSON** на новом.
