# LifeOS PWA

Локальное PWA-приложение для GitHub Pages: работа, день, спорт, привычки, фокус, финансы, проекты, отчёты и бесконечные достижения.

## Что нового в этой версии

- Первый запуск теперь открывает мастер настройки.
- Можно выбрать рабочий период перед началом использования.
- Можно настроить график по каждому дню недели: рабочий/выходной, начало и конец.
- Можно задать желаемое количество тренировок в неделю.
- График можно изменить позже в настройках без потери записей.
- План работы, календарь, отчёты, финансы и антихаос пересчитываются по выбранному графику.
- Ачивки стали бесконечными: новые рубежи генерируются автоматически по часам работы, полным дням, спорту, фокусу, заметкам, проектам и релизам.
- Улучшена адаптация под iPhone PWA: `viewport-fit=cover`, safe-area для челки/Dynamic Island, нижняя мобильная навигация, увеличенные touch-targets, `font-size:16px` у полей для защиты от авто-зума iOS.
- Добавлены новые анимации карточек, прогресса, таймлайна и активного статуса.
- Исправлена логика хранения: LifeOS проверяет `localStorage`, затем использует резервный вариант (`sessionStorage`/cookie/временная память). Старое предупреждение больше не появляется без причины.

## Размещение на GitHub Pages

1. Распакуй архив.
2. Загрузи все файлы в корень репозитория.
3. Включи GitHub Pages: `Settings → Pages → Deploy from branch`.
4. Выбери ветку `main` и папку `/root`.
5. Открой опубликованный URL.

## Установка на iPhone

1. Открой сайт в Safari.
2. Нажми «Поделиться».
3. Выбери «На экран Домой».
4. Нажми «Добавить».

## Хранение данных

Приложение не использует сервер и базу данных. Все записи хранятся в браузере пользователя. Для безопасности есть экспорт JSON и импорт JSON.

Если браузер блокирует `localStorage`, LifeOS попробует использовать резервное хранилище. При ограниченном режиме приложение покажет предупреждение и предложит сделать JSON-бэкап.

## Сброс старого кэша

Если GitHub Pages продолжает показывать старую версию:

1. Открой LifeOS.
2. Перейди в `Настройки`.
3. Нажми `Очистить кэш PWA`.
4. Перезагрузи страницу.
