# LifeOS PWA

LifeOS — статическое PWA-приложение для GitHub Pages. Оно ведёт рабочий график с 3 июля 2026 по 31 августа 2026:

- будни: 10:00–17:00;
- суббота: 10:00–14:00;
- воскресенье: выходной.

## Что есть внутри

- локальное хранение через `localStorage`;
- офлайн-режим через Service Worker;
- календарь рабочего периода;
- журнал интервалов времени;
- кастомный вариант времяпрепровождения через пункт «Другое»;
- привычки, XP, уровни, серия дней;
- экспорт и импорт JSON;
- темы: светлая, тёмная, системная;
- PWA-иконки для iOS, Windows, macOS и Android.

## Публикация на GitHub Pages

1. Загрузи файлы из архива в корень репозитория.
2. Открой `Settings → Pages`.
3. Выбери `Deploy from a branch`.
4. Branch: `main`, Folder: `/root`.
5. Открой опубликованную ссылку.

## Файлы

- `index.html` — структура приложения;
- `styles.css` — визуал и адаптивность;
- `app.js` — логика, календарь, localStorage;
- `manifest.webmanifest` — PWA-манифест;
- `sw.js` — service worker;
- `icons/` — иконки приложения.
